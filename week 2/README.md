# Week 2 Activity 3:OOP Calculator

# Description
Basic Mathematical operations using Object Oriented Programming(OOP) concept.

## OOP Concepts Used
1. **Class**: `Calculator` class is created to define operations
2. **Functions**: display_menu(), main()
3. **Methods**: `add()`, `subtract()`, `multiply()`, `divide()`
    
# How to Run
1. Open the file VS Code
2. Run the Program
3. Choose operation from the menu
4. Enter first and second number
